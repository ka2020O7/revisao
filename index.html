HTML

HTML é a linguagem de marcação para criar páginas web.

Elementos importantes:

  • <html> - Define o documento HTML
  • <head> - Contém metadados
  • <body> - Corpo do documento
  • <div> - Container genérico

CSS

CSS é usado para estilizar elementos HTML.

Propriedades básicas:

  • color - Cor do texto
  • font-size - Tamanho da fonte
  • margin - Espaçamento externo
  • padding - Espaçamento interno

Git e GitHub

Git é como um "controlador de versões" para seus projetos de programação
GitHub é como uma "rede social para códigos" que usa o Git é como se fosse um "Google Drive" especial para programadores, onde além de armazenar seus arquivos, você pode ver o histórico de todas as mudanças e trabalhar com outras pessoas.

Comandos básicos:

  1. git init - Inicia um repositório
  2. git add . - Adiciona arquivos
  3. git commit -m "mensagem" - Salva alterações
  4. git push - Envia para o GitHub
